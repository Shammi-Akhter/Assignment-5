Question-1: What are the different ways to select an element in the DOM?

You can select elements in the DOM using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName(), childNodes, parentNode, and closest().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML content (including HTML tags), innerText gets/sets visible text (excluding hidden text), and textContent gets/sets all text (including hidden text).

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events for its child elements, improving performance and simplifying code.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates up to its ancestors in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create elements with document.createElement(), add them using appendChild() or insertBefore(), and remove them with removeChild() or remove().
