Question-1: What are the different ways to select an element in the DOM?

These are the different ways to select elements in the DOM using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName(), childNodes, parentNode, and closest().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML content (including HTML tags). innerText gets/sets visible text (excluding hidden text).And textContent gets/sets all text (including hidden text).

Question-3: What is event delegation in the DOM?

A single event listener is added to a parent element to handle events for its child elements is Event delegation.

Question-4: What is event bubbling in the DOM?

An event starts from the target element and propagates up to its parents in the DOM hierarchy is Event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements with document.createElement(), add them using appendChild() or insertBefore(), and remove them with removeChild() or remove().
